ASC Portal Data Entry Guide

Before You Start:
  • Open your ASC portal in another tab.
  • Keep your transcript/grade cards ready.
  • Plan to spend about 10-15 minutes for complete data entry.

Features

  • Edit in Place: Direct editing of fields with instant content updates.
  • Dynamic Tables: Capability to add new rows to CPI tables and course tables and semester tables.
  • Automatic Saving: Your data gets saved automatically to your browser when you click "OK".
  • Screenshot Capture: Option to download the whole page as a PNG image. This can be accessed at the bottom of the page after confirming all the changes and saving.
  • Reset Option: Reset all values to their initial state with just one click.

How to Use the Website: Step-by-Step Guide

This tool lets you enter all your transcript information in one single page and save it to your computer as a screenshot.

1. Personal Information Section

Basic Details Entry

  • Look for fields marked with Edit buttons.
  • Click Edit, the field will be empty and ready to take input.
  • Enter your details in the field.
  • Click Ok to save each field.

2. Academic Performance Section

Adding Semester Results

  • Use New CPI Row to add each semester's performance.
  • Enter in chronological order (oldest to newest).
  • All the fields can be manually typed, they are not calculated automatically.

3. Course Details Section

Managing Semester Tables

  • Click add 4th semester courses table (or relevant semester) to add new semester.
  • Use Add one more row if you have additional courses.
  • Use Remove one row to delete extra rows (if added unintentionally).

4. Saving and Managing Your Data

Data Management Features

  • Auto-Save: Your entries are automatically saved in your browser when you click "OK".
  • Confirm: After you are done with all edits, click Confirm All Changes & Save when finished. This will remove the editing options.
  • Screenshot: After confirming all changes, you will get an option to save a PNG image of your completed form.
  • Reset: Use Reset All Values to start over if needed.
  • Scroll To Bottom: You can access the bottom buttons by clicking the down arrow button in the welcome popup message. This button is also used to generate the screenshot.

Troubleshooting

  • Use the browser's refresh button if any buttons become unresponsive.
  • Make sure to save a copy of the final screenshot for your records by clicking the screenshot button after pressing confirm all changes & save.
  • If you need to make changes after confirming, you'll need to start over (press reset and refresh the page).
  • If you are unable to edit any fields (after clicking on reset all values), just change the window by doing alt+tab and then come back to the page, and then edit again.
